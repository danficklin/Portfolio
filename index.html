My projects

Here is a sample of a few of my past projects.

CanvassPlan

My final project in the Eleven Fifty Academy Software Development course was a planning tool that helps political organizers assign volunteers and staff to sites for political canvassing and other ground-game operations, taking into account things like drivers, team leaders, and transportation availability.

Github repository

Who's that Pokémon?


This game was also created as a project at Eleven Fifty Academy, and uses the PokéAPI to randomly generate the silhouette of one of the various pokémon, allowing the user to try to guess which one it is before revealing it.


Github page
Github repository

Pendragon Character Generator

This is a personal project I worked on after finishing at Eleven Fifty Academy, which allows the user to generate a family history and character for the tabletop RPG Pendragon, which involves a complicated system for generating your character's background that is made easy with this app.

Github repository

Mercury



This is a text-based roleplaying game that allows the player to travel through space, exploring moons and wrecked ships, and avoiding or fighting space pirates and enemy aliens.



Github repository

Dice Roller

This app allows the user to generate ability scores for games like Dungeons and Dragons or Pathfinder using a variety of common methods.

Github repository

The Korvosan Matrimony

This is an adventure I wrote for my tabletop RPG group for our homebrew Pathfinder game. The game is set in the Pathfinder Campaign Setting so many of the names are copyrighted material belonging to Paizo Inc., and this adventure is obviously not for commercial distribution, but it gives an example of my writing style in that format.

Adventure!

Loremasters of Pathfinder

This is a playlist that contains several of the videos I have recently been writing, producing, voicing, and editing regarding the lore of the Pathfinder Campaign Setting, for a channel I call Loremasters of Pathfinder. The videos are currently unlisted as they are mainly for my friends, but I may publish them at some point.

Videos!
